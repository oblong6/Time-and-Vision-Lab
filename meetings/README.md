# Meetings

Meetings are stored by year, then by meeting folder:

`meetings/YYYY/YYYY-MM-DD_Day_HHMM/`

Example:
`meetings/2026/2026-02-16_Mon_1200/`

## Start here

- Use the template for new meetings: `meetings/_TEMPLATE/`
- Each meeting folder should include:
  - `README.md` (summary record and links)
  - `slides/` (presentations)
  - `notes/` (minutes, discussion notes, transcripts if available)
  - `files/` (supporting material)

## Naming conventions

### Meeting folder
`YYYY-MM-DD_Day_HHMM` (UK local time)

### Files
`YYYY-MM-DD_topic_presenter.ext`

Examples:
- `2026-02-16_scanpath_similarity_liam.pptx`
- `2026-02-16_methods_roundtable_milena.pdf`
